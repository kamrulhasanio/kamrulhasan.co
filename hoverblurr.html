HoverBlurr

Experience a cleaner, more focused browsing with HoverBlurr. This extension automatically blurs images on any webpage, helping you stay focused by reducing visual distractions. Simply hover over an image to see it clearly.

Key Features

  • Dynamic Blurring: Blurs images on all web pages to minimize distractions.
  • Hover to Reveal: Instantly see images in full clarity by hovering over them.
  • Customizable Blur: Adjust the blur intensity easily through the extension's popup.
  • Whitelist Functionality: Add specific websites to a whitelist to prevent blurring.
  • Simple Interface: Manage settings and whitelisted sites with ease.

Install HoverBlurr in Chrome

To install HoverBlurr in Chrome, follow these steps:

  1. Download the extension ZIP file from the provided link.
  2. Go to chrome://extensions/ in your Chrome browser.
  3. Enable "Developer mode" using the toggle switch in the top-right corner.
  4. Click "Load unpacked" and select the extracted extension folder.
  5. The HoverBlurr extension will be installed and activated.
